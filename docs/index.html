Why PhishScope?

Built for security researchers who need evidence, not just verdicts

Evidence-Driven

Every finding includes concrete evidence and context. No black-box decisions.

Automated Analysis

Headless browser automation with anti-detection measures for realistic page behavior.

JavaScript Inspection

Detects credential theft patterns, event listeners, and data exfiltration code.

Network Monitoring

Captures all traffic and identifies suspicious exfiltration endpoints.

DOM Analysis

Detects forms, hidden inputs, overlays, and suspicious page structures.

Investigation Reports

Human-readable Markdown reports with all artifacts and evidence.

Get Started in 60 Seconds

Install and run your first analysis

1

Install Dependencies

git clone https://github.com/yourusername/PhishScope.git
cd PhishScope
pip install -r requirements.txt
playwright install chromium
2

Run Analysis

python phishscope.py analyze https://suspicious-site.com
3

Review Report

cat reports/case_TIMESTAMP/report.md

# PhishScope Investigation Report

**Target URL:** `https://secure-login-verify.tk/microsoft/login.php`

## Executive Summary

This page exhibits multiple phishing indicators including credential 
collection forms, suspicious JavaScript behavior, and data exfiltration 
endpoints. **HIGH CONFIDENCE** phishing attack.

## Key Findings

### DOM Analysis
- ✓ Found 1 form with password field
- ✓ Form has no action (JavaScript-based submission)
- ✓ Detected 3 hidden input fields

### JavaScript Analysis
- ⚠ Detected input event listeners on password field
- ⚠ Found fetch() POST request to suspicious domain
- ⚠ JSON.stringify() used on form data

### Network Traffic
- 🚨 POST to api.phish-collector.tk (Suspicious TLD)
- 🚨 Direct IP connection: 185.234.219.45:8080
